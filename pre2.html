What I like about Python:

  1. Python is big in open source. Python recently passed Java as #2 most popular programming language on Github (by number of active repositories).
  2. Python is currently one of the fastest growing programming languages, according to Stack Overflow's Developer Survey (and other sources).
  3. The same survey rated Python one of the best in terms of developer experience.
  4. Python's Django web framework is designed for rapid development. Django makes unimportant decisions for you so you can focus more on the big picture.
